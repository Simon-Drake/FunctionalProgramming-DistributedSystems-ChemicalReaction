**chemistry.erl** is a program that is supposed to simulate a chemical reaction by spawning parallel processes that communicate with eachother. In this case when two molecules collide they may react. 

**Requirements** : 
- Erlang Installed

**To run:**

Run Erlang Shell:

*erl* 


Compile : 

*c(chemistry).*


Call React :

*chemistry:react(2,3).*
